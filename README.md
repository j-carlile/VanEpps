# VanEpps Project

Computational workflow for analyzing bacterial FASTA files and developing primer-selection methods.

## Folder structure
- `data/raw_fastas/`: local raw FASTA files, not tracked by Git
- `data/references/`: metadata and reference tables
- `data/processed/`: processed outputs, not tracked by Git
- `scripts/`: Python scripts
- `notebooks/`: exploratory analyses
- `results/`: summary outputs, usually not tracked unless small/final
